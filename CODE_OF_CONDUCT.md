# Code of Conduct

## Our Commitment
Nos comprometemos a mantener una comunidad abierta, respetuosa y colaborativa para todas las personas.

## Expected Behavior
- Trato respetuoso y profesional.
- Feedback tecnico claro, sin ataques personales.
- Disposicion a ayudar y documentar decisiones.

## Unacceptable Behavior
- Acoso, insultos o discriminacion.
- Publicacion de informacion privada sin permiso.
- Conductas que deterioren deliberadamente el entorno colaborativo.

## Enforcement
Reportes al mantenedor principal del repositorio via issue privado o contacto directo en GitHub.
