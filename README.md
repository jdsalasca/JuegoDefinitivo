# JuegoDefinitivo

Plataforma educativa en Java que convierte libros en aventuras jugables.

## Modulo activo
- `Juego/Elcaballero`: motor E2E `txt/pdf -> escenas -> juego`.

## Inicio rapido
```bash
cd Juego/Elcaballero
mvn test
mvn exec:java
```
