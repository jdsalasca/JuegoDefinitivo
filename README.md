# JuegoDefinitivo

Plataforma educativa que convierte libros en aventuras jugables.

## Monorepo
- `apps/backend`: motor y API del juego.
- `apps/frontend`: interfaz React.
- `docs`: arquitectura, roadmap y guias.

## Inicio rapido
```bash
cd apps/backend
mvn test
mvn exec:java
```
